### 功能

✅ 适配 PC & 移动端

✅ 文章分类 - 识别 md 的中的 tag, 自动分类

✅ 评论功能 - gitalk

<br>

### 技术栈 :wrench:

- [vitepress](https://vitepress.vuejs.org/)
- [tailwindcss](https://www.tailwindcss.cn/docs)

<br>

### 跑起来 🏃‍♂️

```sh
# 安装依赖
yarn

# 运行
yarn docs:dev app
```
